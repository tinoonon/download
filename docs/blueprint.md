# **App Name**: DevLab: FiveM Studio

## Core Features:

- Service & Plan Showcase: Display detailed FiveM development services, multi-tiered plans (Starter, Pro, Full) with pricing, inclusions, and a visual comparison for clarity.
- Discord Lead Generation: Integrate prominent calls to action (CTAs) that directly link users to the DevLab Discord server for consultations, project discussions, and hiring developers.
- User Testimonials: Present a dedicated section featuring client testimonials (name, server, short comment, star rating) to build trust and credibility for services offered.
- Technology Stack Showcase: Highlight the expertise in relevant FiveM technologies (e.g., VRP, VRPEX, Creative, Lua, MySQL, APIs) through an engaging section with neon-styled icons.
- Interactive UI Elements: Implement visually dynamic UI elements such as card hover effects, glowing borders, smooth section fade-ins, and button ripple/glow animations for a modern, premium feel.
- Responsive & Optimized Experience: Ensure a fully responsive design for seamless navigation and optimal content display across all devices, including desktops, tablets, and mobile phones, with a focus on performance.
- AI-Assisted Project Briefing Tool: Provide a generative AI tool to assist clients in clearly articulating their FiveM server development requirements, creating a structured project brief that streamlines the inquiry process via Discord.

## Style Guidelines:

- Background color: Deep, almost black (#0A0A0A) to establish a dark, sophisticated, and futuristic canvas.
- Primary color: A vibrant neon blue (#0D8CF2) for headings, active elements, and interactive components, providing a bold contrast against the dark background.
- Accent color: A subtle neon turquoise/cyan (#1E998E) used for secondary highlights, subtle glows, and complementary visual effects.
- All text uses the 'Space Grotesk' sans-serif font for its clean, modern, and techy aesthetic, aligning with the futuristic/cyberpunk theme.
- Custom vector icons featuring neon glow effects, particularly for services, technology showcase, and prominent UI elements like the Discord button, ensuring visual consistency with the dark theme.
- Minimalist and well-spaced interface with a fixed header for navigation, modern glass-effect cards with neon borders for plans and services, and a persistent floating Discord button for easy access.
- Subtle, elegant animations including hover-triggered glows on buttons, gentle fade-in effects for sections as the user scrolls, slight scale changes on cards, and an animated background gradient or particles for an immersive feel. The 'Contratar' buttons feature a prominent ripple or expanding glow effect upon interaction.
